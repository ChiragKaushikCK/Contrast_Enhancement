## ‚û§ Project Overview

This project demonstrates **contrast enhancement** techniques using the Python Imaging Library (**PIL**) and visualizes results using **Matplotlib**.

‚û§ The program loads an image, enhances its contrast with varying levels, and displays both the **original** and **enhanced images** side-by-side.

‚û§ Ideal for beginners in image processing, this notebook is a great hands-on exercise to understand visual contrast manipulation and Python-based graphics rendering.

---

## ‚û§ Features Implemented

‚úî Load and visualize images with Matplotlib
‚úî Apply contrast enhancement using `ImageEnhance.Contrast()`
‚úî Adjust contrast dynamically from `1x` to `9x`
‚úî Display results side-by-side using subplots
‚úî Compare original vs. enhanced versions intuitively

---

## ‚û§ How It Works

‚û§ Import libraries: `PIL`, `ImageEnhance`, and `Matplotlib`
‚û§ Load an image using `Image.open()`
‚û§ Enhance contrast using `enhancer.enhance(factor)`
‚û§ Iterate through various enhancement levels (e.g., 1 to 5 or 1 to 9)
‚û§ Use `matplotlib.pyplot` to visualize each comparison
